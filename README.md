# Kurshantering med TypeScript

## Beskrivning

Detta projekt är en webbapplikation utvecklad med TypeScript och Vite. Syftet med applikationen är att hantera kursinformation genom att låta användaren lägga till, visa och radera kurser.

Varje kurs innehåller följande information:

- Kurskod
- Kursnamn
- Progression (A, B eller C)
- Länk till kursplan

Kursinformationen sparas i webbläsarens localStorage, vilket gör att informationen finns kvar även efter att sidan har laddats om eller webbläsaren har stängts.

## Funktioner

- Lägga till nya kurser
- Visa en lista över sparade kurser
- Radera kurser
- Validering av unik kurskod
- Validering av progression (A, B eller C)
- Lagring av data med localStorage
- Responsiv design för både dator och mobil

## Tekniker

Projektet är utvecklat med:

- HTML
- CSS
- TypeScript
- Vite
- LocalStorage
